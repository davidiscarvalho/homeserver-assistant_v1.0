# Secrets (Gitignored)

This folder holds local secrets and credentials.

## Files
- `example.env` (committed): example keys and expected variables
- `openclaw.env` (gitignored): your real secrets

## Usage
1. Review `secrets/example.env`
2. Create `secrets/openclaw.env` with real values.
3. Export variables before running backup/restore/projections if needed.
